**What is Billify?**

Billify is a simple yet powerful supermarket billing software built with Python and Tkinter. It offers an easy-to-use interface for managers, streamlining the billing process and inventory management.


**Key Features:**
  -->GUI-based manager interface:**
    * Secure login for authorized personnel.
    * Create new bills for customers.
    * Add new products with details and manage existing ones.
    * Edit product quantities and prices.
    
  -->Seamless customer billing:**
    * Choose new or existing customer.
    * Enter details for new customers.
    * Automatically fetch information for existing customers using phone number.
    * Real-time product quantity updates reflect purchases.
    
  -->Database-driven:**
    * Leverages MySQL for secure data storage and management.
    * Efficient record keeping and easy data retrieval.



**Contributing:**

We welcome contributions to Billify! Feel free to fork the repository, make changes, and submit pull requests.
